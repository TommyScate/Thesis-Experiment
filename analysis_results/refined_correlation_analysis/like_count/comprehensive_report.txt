## 📋 Methodology Notes

**Statistical Methods:**
- Spearman ρ: Robust to outliers, captures monotonic relationships
- Log transformations: log(x+1) for count variables
- Percentile rankings: Rank-based transformations
- Point-biserial: Correlations with binary success thresholds
- Mann-Whitney U: Non-parametric group comparisons

**Effect Size Interpretations:**
- Negligible: |ρ| < 0.2
- Small: 0.2 ≤ |ρ| < 0.5
- Medium: 0.5 ≤ |ρ| < 0.8
- Large: |ρ| ≥ 0.8
